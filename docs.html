System Requirements

Minimum requirements for our games:

  • OS: Windows 10+ / macOS 10.15+ / Ubuntu 18.04+
  • RAM: 4GB minimum, 8GB recommended
  • Graphics: DirectX 11 compatible
  • Storage: Varies by game (typically 2-10GB)

API Documentation

Technical documentation for developers integrating with Steel Canvas Studio services.

Authentication

All API requests require authentication using API keys or OAuth 2.0 tokens.

Endpoints

  • GET /api/games - List available games
  • GET /api/user/profile - User profile information
  • POST /api/user/achievements - Update achievements
  • GET /api/leaderboards - Game leaderboards

Rate Limits

API calls are limited to 1000 requests per hour per API key.

SDKs and Libraries

Official SDKs available for:

  • Unity (C#)
  • Unreal Engine (C++)
  • JavaScript/Node.js
  • Python

Troubleshooting

Common Issues

  • Game won't start: Verify system requirements and update drivers
  • Performance issues: Lower graphics settings and close background apps
  • Connection problems: Check firewall settings and internet connection
  • Save game issues: Verify file permissions and available storage
