Steel Canvas - From Solo Dev to AAA Dreams

FEATURED SUPPORT

Currently developing: Pocket Legion

Hey, I'm the solo developer behind Steel Canvas. Right now it's just me in my apartment, fueled by coffee and a dream to build something bigger. My goal? Start with mobile games that actually respect players, then grow into the AAA studio I've always envisioned.

The Journey So Far

Steel Canvas began when I got tired of seeing the same predatory monetization in every mobile game. As a gamer first and developer second, I knew there had to be a better way. So I quit my day job, taught myself game development, and started building the games I wanted to play.

Pocket Legion is my first commercial project - a strategic crowd runner that proves mobile games can have depth without dark patterns. It combines the instant fun of hypercasual games with actual strategic gameplay, wrapped in internet culture that doesn't talk down to players.
