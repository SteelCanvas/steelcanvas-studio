Security Policy

Last updated: June 10, 2025

Our Commitment to Security

At Steel Canvas Studio, we take security seriously. We are committed to protecting our users' data and maintaining the integrity of our games and services.

Responsible Disclosure

We encourage responsible disclosure of security vulnerabilities. If you discover a security issue, please report it to us privately before public disclosure.

How to Report

Send security reports to: security@steelcanvas.studio

What to Include

  • Detailed description of the vulnerability
  • Steps to reproduce the issue
  • Potential impact assessment
  • Any supporting screenshots or proof-of-concept
  • Your contact information for follow-up

Our Response Process

  1. Acknowledgment - We will acknowledge receipt within 24 hours
  2. Assessment - Initial assessment within 72 hours
  3. Updates - Regular updates throughout the investigation
  4. Resolution - Fix deployment and coordinated disclosure

Security Measures

Data Protection

  • Encryption of data in transit and at rest
  • Regular security audits and penetration testing
  • Access controls and authentication systems
  • Monitoring and logging of security events

Game Security

  • Anti-cheat systems and fair play enforcement
  • Secure multiplayer architectures
  • Regular updates and security patches
  • Player reporting and moderation systems

Infrastructure Security

  • Secure hosting and cloud infrastructure
  • DDoS protection and mitigation
  • Regular backup and disaster recovery procedures
  • Network security monitoring

User Security Guidelines

Account Security

  • Use strong, unique passwords
  • Enable two-factor authentication when available
  • Keep your account information up to date
  • Don't share your login credentials

Safe Gaming Practices

  • Download games only from official sources
  • Keep your gaming client updated
  • Be cautious of suspicious links or downloads
  • Report cheating or suspicious behavior

Incident Response

In the event of a security incident, we will:

  • Immediately assess and contain the threat
  • Notify affected users within 72 hours when required
  • Provide clear information about the incident
  • Implement necessary security improvements
  • Cooperate with law enforcement when appropriate

Third-Party Security

We work with trusted third-party providers who meet our security standards. All partners undergo security assessments and must comply with our security requirements.

Security Updates

We regularly update our security measures and policies. Critical security updates for our games and services are deployed immediately.

Contact Information

For security-related inquiries:

  • Security issues: security@steelcanvas.studio
  • General questions: contact@steelcanvas.studio
  • Administrative matters: admin@steelcanvas.studio
